# Quick Repay

A simple 2-page loan repayment site using TailwindCSS.  
Upload to GitHub Pages to host.

## Pages
- `index.html` - Loan Products
- `repay.html` - UPI Copy Page

## How to Host
1. Create a new public repository on GitHub (example: quick-repay)
2. Upload all files
3. Go to Settings > Pages > Deploy from branch (main, root)
4. Visit: https://your-username.github.io/quick-repay/
